# PIXEL_HAVEN // ARCADE_OS

![Favicon](assets/images/icons8-pixel-star-16.png)

A high-performance, retro-arcade browser hub featuring a collection of pixel-art games with immersive CRT visual effects and a sleek cyberpunk aesthetic.

## 🚀 Overview

**Pixel Haven** is a centralized arcade platform built with modern web technologies. It provides a cohesive experience for playing multiple retro-styled games through a unified system called **ARCADE_OS**.

## 🕹️ Game Library & How to Play

### 1. NEON//RUN
*NEON BLADE SEQUENCING*
- **Objective**: Sprint through the neon landscape and jump over obstacles.
- **Controls**:
  - **PC**: `SPACE` to Jump | `SHIFT` for Slow-Mo | `ESC` to Pause
  - **Mobile**: `TAP` to Jump | On-screen button for Slow-Mo

### 2. MEM//FRAGMENT
*MEMORY LEAK CRISIS*
- **Objective**: Replicate the blinking sequence shown by the grid.
- **Controls**:
  - **PC/Mobile**: `CLICK` or `TAP` tiles to repeat the sequence.
  - **PC**: `ESC` to Pause

### 3. ECHO//LOOP (ECHO//RUN)
*TEMPORAL INPUT FEEDBACK*
- **Objective**: Climb as high as possible. Beware: every 3.5s, your recent past movements replay!
- **Controls**:
  - **PC**: `A`/`D` to Move | `SPACE` to Jump
  - **Mobile**: Use on-screen directional and jump buttons.

### 4. CLASH//CORE
*KINETIC STRIKE PROTOCOL*
- **Objective**: Time your strikes perfectly with the sync pulse to deal damage.
- **Controls**:
  - **PC**: `SPACE` or `CLICK` to Clash
  - **Mobile**: `TAP` to Clash

### 5. LOCK//AIM
*PRECISION DIRECTIONAL BATTLE*
- **Objective**: Eliminate enemies with precise aimed shots. One shot at a time!
- **Controls**:
  - **PC**: `MOVE MOUSE` to Aim | `HOLD CLICK` to Lock | `RELEASE` to Fire
  - **Mobile**: `LEFT SIDE` of screen to Aim | `RIGHT SIDE HOLD` to Lock | `RELEASE` to Fire

### 6. RICO//RUSH
*REFLECTIVE COMBAT SPEED*
- **Objective**: Eliminate all enemies with a single bouncing kinetic shot.
- **Controls**:
  - **PC/Mobile**: `DRAG` to Aim | `RELEASE` to Fire

### 7. PULSE//SURVIVE
*KINETIC TIME CONTROL SURVIVAL*
- **Objective**: Survive as long as possible. Note: Time only flows when you move!
- **Controls**:
  - **PC**: `WASD` to Move
  - **Mobile**: Use the on-screen virtual joystick to Move

### 8. NULL//PAUSE
*TEMPORAL DRIFT RACING*
- **Objective**: Drive through sectors and use your "Null State" to avoid danger.
- **Controls**:
  - **PC**: `WASD` to Drive | `SPACE` for Null State
  - **Mobile**: Virtual Joystick to Drive | `NULL` button for Null State

### 9. BRICK//BREACH
*HARDPOINT DESTRUCTION*
- **Objective**: Classic brick-breaking action. Clear the sector to proceed.
- **Controls**:
  - **PC**: `MOUSE` to Move Paddle | `SPACE` to Launch Ball
  - **Mobile**: `TOUCH & DRAG` to Move Paddle | `TAP` to Launch Ball

### 10. MEMORY//GLITCH
*NEURAL DECEPTION PROTOCOL*
- **Objective**: Match all pairs before the fail limit triggers a grid shuffle.
- **Controls**:
  - **PC/Mobile**: `CLICK` or `TAP` to flip tiles.
  - **Hold**: `HOLD` for 3s on disguised tiles to reveal their true identity.


## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Logic**: Vanilla JavaScript (ES6+ Modules)
- **Styling**: Modern CSS3 with Custom Properties (Variables)
- **Aesthetics**: 
  - CRT Scanlines & Flicker effects
  - Dynamic HSL color palettes
  - CSS Grid & Flexbox layouts
  - responsive design for Desktop & Mobile


## 🔋 Getting Started

To launch the Arcade Hub:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Use your mouse, keyboard, or touch inputs to navigate the game selector.
